## ?? KEY ALGORITHMS IMPLEMENTED

### Text Featurization (ML.NET)
```csharp
Transforms.Text.FeaturizeText()
?? Tokenizes text into words
?? Applies TF-IDF weighting
?? Outputs 50-dimensional vector
```

### Cosine Similarity
```
similarity = (V? · V?) / (||V?|| × ||V?||)
where:
  V? = Query vector
  V? = Student vector
  Range: 0 (different) to 1 (identical)
```

### Grade Filtering
```csharp
IsPassed() = (Grade == 'A' || Grade == 'B')
```

### Subject Filtering
```csharp
HasScience() = Contains("physics" || "chemistry" || "biology" || "science")
```
